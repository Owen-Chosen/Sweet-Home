Global Medical Consultation Platform — Backend System

A scalable backend service that enables people anywhere in the world to instantly connect with verified medical specialists. 
The system manages patient–doctor matching, real-time appointment scheduling, and global user onboarding. 
Built with microservices architecture, async request processing, and robust data validation, this project showcases strong backend engineering practices while 
supporting a mission-critical health experience that works reliably across regions and time zones.
